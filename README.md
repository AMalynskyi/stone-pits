# StonePits
Game with Stones &amp; Pits

## Used libraries and engines 

1) LibGDX - core game engine
2) Artemis ODB - entities engine
3) VisEditor - scene design
4) JUnit, AssertJ - testing
5) Java 8 
6) Gradle - build and assembling
